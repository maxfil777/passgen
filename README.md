# PassGen Clone (V2)

Минималистичный генератор паролей на `HTML + JavaScript` с упором на безопасность:
- криптостойкая генерация через `Web Crypto API` (`crypto.getRandomValues`);
- проверка пароля на утечки через [Have I Been Pwned](https://haveibeenpwned.com/) по методу **k-anonymity** (без передачи самого пароля).

## Функционал

- Генерация пароля с настройками:
  - цифры `0-9`;
  - маленькие буквы `a-z`;
  - большие буквы `A-Z`;
  - спецсимволы `%*)?@#$~`;
  - режим без повторяющихся символов;
  - длина: `6, 8, 10, 12, 14, 16, 24, 32, 40, 48, 56, 64`.
- Копирование пароля в буфер обмена.
- Индикатор надежности пароля.
- История последних 10 паролей.
- Автогенерация при изменении параметров.
- Проверка утечек через HIBP:
  - локально считается `SHA-1` хеш пароля;
  - в сеть отправляются только первые 5 символов хеша;
  - сравнение полного хеша выполняется в браузере.

## Безопасность

- Пароль генерируется через криптостойкий источник случайности (`crypto.getRandomValues`).
- При проверке утечек пароль **не отправляется** на сервер.
- Используется подход k-anonymity API HIBP (`/range/{prefix}`).

## Запуск локально

Откройте `index.html` в браузере.

Рекомендуется запускать через локальный сервер:

```bash
# Python 3
python -m http.server 8080
```

После этого откройте: `http://localhost:8080/passgen-clone/`

## Тесты

Тесты написаны на `node:test` и проверяют:
- SHA-1 хеширование;
- разбор ответа HIBP;
- корректный вызов HIBP API с 5-символьным префиксом;
- сценарии найденной/не найденной утечки;
- обработку ошибок сервиса.

Запуск:

```bash
node --test ./passgen-clone/script.test.js
```
